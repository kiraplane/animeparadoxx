# Anime Paradox X Wiki

English Roblox guide site for [animeparadoxx.wiki](https://www.animeparadoxx.wiki).

The site covers current codes, units, tier decisions, traits, evolution, Etherealize, materials, team building, game modes, verified links, and re-release guides. The hero remains full width and the wiki sidebar begins immediately below it.

## Local development

```bash
pnpm install
pnpm dev
```

Validate with `pnpm lint`, `pnpm exec tsc --noEmit`, `pnpm next:build`, and `pnpm build`.
